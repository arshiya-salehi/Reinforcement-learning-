# Project Submission Package

This directory contains the submission package for CS 175 Final Project.

## Files Description

### Main Files
- `project.ipynb` - Jupyter notebook demonstrating the project (runs in <1 minute)
- `project.html` - HTML export of the notebook with all outputs
- `README.md` - This file, containing one-line descriptions of all files

### Source Code Directory (`src/`)
- `__init__.py` - Package initialization file
- `config.py` - Centralized configuration for training, evaluation, and hyperparameters
- `load_generators.py` - Implementations of load generation patterns (steady, sinusoidal, spikes, Poisson)
- `train_dqn_sb3.py` - Training script for DQN agent using stable-baselines3
- `train_ppo_sb3.py` - Training script for PPO agent using stable-baselines3
- `hpa_autoscaler.py` - Kubernetes HPA-style rule-based autoscaler implementation
- `static_autoscaler.py` - Static baseline autoscaler with fixed instance count
- `env_wrapper.py` - Environment wrapper to fix observation space compatibility with stable-baselines3
- `evaluate.py` - Main evaluation script to compare all methods (RL agents, HPA, static)
- `metrics.py` - Metrics collection and statistical analysis utilities
- `visualize.py` - Visualization scripts for generating comparison plots

### Models Directory (`models/`)
- Contains pre-trained RL models (DQN and PPO) saved as .zip files
- Models are loaded by the notebook for demonstration

### Data Directory (`data/`)
- Contains sample data files if needed (total size <5MB)
- Currently empty as the project uses synthetic data generated by the environment

## How to Run

### Prerequisites
1. Install required packages: `pip install -r requirements.txt`
2. Install the gym_scaling package: `pip install -e .`

### Running the Notebook
1. Open `project.ipynb` in Jupyter Notebook or JupyterLab
2. Run all cells (should complete in <1 minute)
3. The notebook will:
   - Load pre-trained models (if available)
   - Run a short demonstration comparing HPA and RL agents
   - Generate visualization plots

### Running Training (Optional)
To train your own models:
```bash
# Train DQN
python src/train_dqn_sb3.py --pattern RANDOM --timesteps 200000

# Train PPO
python src/train_ppo_sb3.py --pattern RANDOM --timesteps 500000
```

### Running Evaluation
To run comparative evaluation:
```bash
python src/evaluate.py --patterns STEADY SINUSOIDAL SPIKE POISSON --episodes 5
```
